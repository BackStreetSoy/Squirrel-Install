# Squirrel-Install

Steps:

1. Clone the Repo

2. In terminal run the following command: 
```bash
chmod a+x squirrel_setup.sh
./squirrel_setup.sh
```

Remember to delete the Repo after you've run the script. Happy Squirrelling!
